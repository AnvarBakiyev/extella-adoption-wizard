# Плагин «Визард внедрения» — обзор

Нативная поверхность Adoption Wizard в тулбаре Extella. Локальный мост на 127.0.0.1:8765
(server.py в ~/extella_wizard/app, UI wizard.html): интервью → blueprint → тест → запуск.
Вкладка «Автоматизации»: блок «Подключения» (5 каналов вывода, 6 источников входа),
карточки процессов с видимой обвязкой (источник/расписание/куда слать/входящие),
запуск сейчас/по расписанию, продовый агент 24/7, панель мониторинга. Deep-link: /?session=<id>.

Методология визарда (интервью с гипотезами, spec-driven, task-breakdown, качество экспертов) —
отдельные концепты, экспортируются из аккаунта при упаковке (см. README_CONCEPTS.md).
