# Концепты — что здесь и что доэкспортировать

- `000_wizard_overview.md` — обзор плагина (готов).

**Доэкспортировать из аккаунта** (один шаг перед публикацией): концепты методологии визарда
(интервью с гипотезами, idea-refine, spec-driven, task-breakdown, §4 качества экспертов) и
каталог возможностей. Они живут в аккаунте Extella как концепты с эмбеддингами. Экспорт:
`concept_search`/`concept_list` → сохранить каждый как `concepts/<id>_<slug>.md` с заголовком
`# concept_id: <id>`. Тело — markdown. При установке (INSTALL.md, шаг 2) они добавятся `concept_add`.
